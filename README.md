# CareerPilot AI

## Overview

CareerPilot AI is an AI-powered career guidance system designed to help students identify suitable career paths, analyze skill gaps, generate personalized learning roadmaps, recommend portfolio projects, and prepare for technical interviews.

This project is being developed as part of the NVIDIA AI GPU Summer Internship Capstone Project.

## Objectives

* Recommend suitable career paths based on student skills and interests.
* Identify missing skills required for target careers.
* Generate structured learning roadmaps.
* Suggest portfolio-worthy projects.
* Provide interview preparation guidance.

## Technology Stack

* Python
* Streamlit
* PyTorch
* Transformers
* Hugging Face
* LoRA (PEFT)
* NVIDIA GPU Acceleration
